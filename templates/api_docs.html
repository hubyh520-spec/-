API对接文档

以下是AC云验证平台提供的API接口，供C++/其他语言客户端对接使用。

基础地址

http://{{ request.host }}/api/
1. 卡密验证接口
POST /api/verify

验证卡密有效性，支持普通卡和设备卡。

请求参数

参数名类型必填说明
card_passwordstring是卡密号码
instance_idstring是实例ID
device_idstring设备卡必填设备唯一标识
user_idstring否用户ID

返回示例

{
    "code": 1,
    "message": "验证成功",
    "card_type": "normal",
    "valid_days": 30,
    "remaining_days": 25,
    "expire_time": 1735689600
}
2. 云更新接口
POST /api/check_update

检查是否有新版本可用。

参数名类型必填说明
user_idstring是用户ID
instance_idstring是实例ID
current_versionstring否当前版本号
GET /api/download/{user_id}/{instance_id}/{version}

下载更新文件，会自动扣除流量。

3. 公告接口
POST /api/get_announcement

获取实例公告内容。

参数名类型必填说明
instance_idstring是实例ID
user_idstring否用户ID
4. 配置获取接口
POST /api/get_config

获取实例的加密配置信息。

参数名类型必填说明
instance_idstring是实例ID
GET /api/get_server_time

获取服务器时间戳。
